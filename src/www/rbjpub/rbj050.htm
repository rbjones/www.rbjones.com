The rbjones.com.tgz file is a compressed tar file which can be unpacked using the Linux tar program or the Windows application Winzip. The file is about 8Mb in size, and when unpacked the web site occupies about 25Mb. When unpacked the entire web site is found in a single directory called www and can be browsed by opening the file www/index.htm using a web browser (not www/index.html; that would send you to the online site).
