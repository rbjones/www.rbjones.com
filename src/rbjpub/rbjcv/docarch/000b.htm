docarch - index of conversion scripts

The documents in this directory come in a variety of source formats are are converted into HTML using PERL scripts.

  • convall.bat - batch conversion of all sources (for which conversions are available)
  • test.bat - run convertor currently under development
  • form00.htm - description of source formats
  • conv01.pl - nroff with defs converter
  • test.pl - converter under development
  • trans1.pl - translator (mainly for unusual characters)
